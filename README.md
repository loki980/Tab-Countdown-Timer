# Tab-Countdown-Timer
Chrome/Edge extension that prompts the user for minutes/seconds and submits a form to start a countdown.  When the countdown reaches zero, the tab closes.

----

## To load an unpacked extension:
https://docs.microsoft.com/en-us/microsoft-edge/extensions-chromium/getting-started/extension-sideloading

## Overview
* **/background/background.js** -- Alarm actions and badge countdown management
* **/popup/popup.html** and **/popup/popup.js** -- Webpage for the form that displays upon clicking the badge
